### The Problem

BlueBubbles Private API is disabled. It's actually a "helper bundle" that requires **SIP (System Integrity Protection) to be disabled** to install.

Per https://docs.bluebubbles.app/private-api/:
- Private API = helper bundle for advanced features (typing indicators, reactions, webhooks, etc.)
- Requires SIP disabled to install
- Can re-enable SIP after installation
- But this is a security trade-off

BlueBubbles app shows:
- ✅ Messages API: Available
- ❌ SIP Disabled: Fail

**Without Private API:** No webhooks, which means no inbound message routing from iMessage.
